# 易截（EasyShot）——适配统信的截屏工具

## 开发目标

开发适配统信的截屏工具，该工具可便捷的使用以下功能：

1. 基础的截屏功能：截取屏幕选定矩形区域
2. 截屏后贴图（添加阴影效果）、保存
3. 贴图后的修改功能：图片缩放以及添加形状、文字、涂鸦（以及擦除）、箭头、马赛克、撤回修改等
4. 历史截屏查看
5. 高级功能：针对窗口截屏、截屏后的文字识别
